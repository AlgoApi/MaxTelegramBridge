# MaxTelegramBridge
Мост-юзербот(UserBot) для Max позволит вам «эмигрировать» в привычный Telegram, сохранив формальное присутствие на навязанной платформе.  Проект зеркалирует сообщения, группы и каналы из Max в ваши приватные чаты Telegram, обеспечивая полную изоляцию от взаимодействия с Max. Max to telegram, bridge, forwarder, пересылка сообщений
